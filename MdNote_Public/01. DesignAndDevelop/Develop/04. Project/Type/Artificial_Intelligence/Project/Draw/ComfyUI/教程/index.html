介绍

Comfy介绍

WebUI和Comfy都是属于StableDiffusion

WebUI

界面：Gradio

优点

  • 要求配置较低效率高 (对显存要求小了很多)
  • 易于管理更灵活、有较强的自组性
  • 未来将与Webui在应用层面拉开距离

缺点

  • 初期不好上手

ComfyUI

界面：图形节点流程图设计

优点

  • 简单易学上手快
  • 固定模板式的界面方便简单的操作

缺点

  • 不易于长期管理
  • 插件较多时过于臃肿
  • 遇到复杂流程时操作费时

很多的节点式软件：Blender、达芬奇、UE

课程介绍

  • 1~6章，基础部分
    • 配置、安装、部署
    • 原理、提示词
    • AIGC网站、模型下载
    • 图生图、高清修复
  • 7~13章，进阶部分
    • Controlnet、诸多处理器的使用方法、场景
    • IPAdapter应用
    • AI视频生成原理
    • AI视频换脸转绘
